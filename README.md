# Mauritius Fuel Prices

Track petrol and diesel retail price changes in Mauritius from 2002 to present.

Data sourced from the [State Trading Corporation (STC)](https://www.stcmu.com/ppm/retail-prices).

## Stack

- [Nuxt 4](https://nuxt.com/) with Vue 3
- Vanilla CSS with custom properties (light/dark theme)
- SVG-based interactive price chart (zero chart dependencies)
- Static site generation for GitHub Pages

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run generate
```

Output is in `.output/public/`, ready for static hosting.
